# SUNROOOF - Luxury Window Experience

A premium, performance-optimized, and fully animated B2B industrial website for **SUNROOOF**, featuring advanced scrollytelling and interactive product showcases.

## 🌟 Features

- **Sunroof Showcase**: High-end frame-by-frame scrollytelling animation (80+ frames) with cascading shutters.
- **French Window Reveal**: A sophisticated scroll-triggered reveal of the French Window line with a "liquid glass" product card.
- **App Control Simulation**: Interactive mobile app mockup that controls the "Skylight" ambiance of the webpage in real-time.
- **Natural Light Engine**: Realistic window-light rays that retract and expand based on the skylight status.
- **Patent Section**: Technical drawing showcase with professional technical SVG animations and luxury documentation access.
- **Smooth Interaction**: Powered by GSAP, ScrollTrigger, and Lenis for ultra-smooth inertia scrolling and masking effects.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System)
- **Animations**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Timeline)
- **Scrolling**: [Lenis](https://github.com/studio-freight/lenis) (Smooth Scroll)
- **Interactive UI**: Custom JavaScript for real-time light/mode simulation

## 📁 Directory Structure

- `website/`: Main project source code (HTML, CSS, JS, Assets).
- `Generated images/`: Reference assets and AI-generated design iterations.
- `Image mockups/`: High-fidelity UI/UX design mockups.
- `Project/`: Project management and additional documentation.

## 🚀 Getting Started

Simply open `website/index.html` in any modern web browser to view the experience.
